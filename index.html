Mechaniki

👊

Walka

Klik = uderzenie. DMG = siła × pety. Brainrot ginie → loot + respawn 5-15s.

🥚

Jajka

10-30% drop chance. Lepsze z trudniejszych kontynentów. Darmowe co 15 min, płatne instant.

🔄

Fuzja

3 identyczne → 1 wyższy tier. Od bazy lvl 3. Mythic fuzja = Shiny variant.

💎

Sloty

3 aktywne pety (base). +3 za Game Pass (199 R$). Baza lvl 7 = 5 slotów.

Retencja

Mechaniki utrzymujące graczy — powody do codziennego wracania

📅

Daily Rewards

7-dniowa seria. Dzień 7 = rzadkie jajko. Reset przy opuszczeniu dnia. Codzienny nawyk.

🏆

Rankingi

Globalny + per kontynent. Korony, aury. Top 3 = bonus do dropów.

🎪

Eventy

Co 2-4 tyg. Limitowany kontynent (Antarktyda, Atlantyda). Unikalne pety, nigdy nie wrócą.

🔑

Kody

YouTube / Discord / TikTok. Drobne nagrody. Organiczny ruch.

🤝

Trading

Wymiana petów. Żywa ekonomia. Pety z różnych kontynentów = różna wartość.

♻️

Rebirth

Reset + trwały ×2. Nowa podróż, prestiżowe efekty. Nieskończony cel.
